Simple AI Chat

A simple chat app built with React + Vite + TypeScript + Tailwind to interact with a local AI using the AnythingLLM API.

  • Modern chat bubble interface, resizable chat area.
  • Dark mode toggle.
  • Shows loaded model and workspace from AnythingLLM.
  • Tool_call support (e.g., current time in CDMX).
  • AnythingLLM token is editable from the UI (no prompt dialogs).
  • Conversation is persistent (saved in localStorage).
  • Compatible with AnythingLLM's /api/v1/openai/chat/completions endpoint.

What is Simple AI Chat?

Simple AI Chat is a web application that allows you to chat with a local AI in a modern, fast, and useful way.

Main features

  • Displays the AI model loaded from LM Studio.
  • Animation "Thinking..." while the AI is responding.
  • Modern, responsive, and easy-to-use interface.

How does it work?

  1. The user types a question or message.
  2. The query is sent to the local AI via LM Studio.
  3. The model responds in real-time, displaying the text as sent by the API.

Technologies used

  • React + Vite
  • TypeScript
  • Tailwind CSS
  • LM Studio (Local API)
